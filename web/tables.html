Tipología logica de red
En base a la solución propuesta y al nivel de seguridad requerido, se propone el diseño de una red con dirección IP clase C, con dos VLAN configuradas de la siguiente manera:
VLAN IP Mascara Broadcast Rango de IPS
1 192.168.0/23 255.255.254.0 192.168.1.255 192.168.0.1 - 192.168.1.254
2 192.168.0/26 255.255.255.192 192.168.2.64 192.168.2.1 - 192.168.2.63
De esta forma, se logrará una comunicación entre equipos de la misma red de forma más segura. Optaremos por utilizar un método de contienda no determinista, donde los dispositivos puedan intentar acceder al medio cada vez que tengan datos para enviar. Para evitar caos completo en los medios, estos métodos usan un proceso de Acceso múltiple por detección de portadora (CSMA) para detectar primero si los medios están transportando una señal.

Las caracteristicas que nos llevan a la elección de este acceso son las siguientes:
  1. Las estaciones pueden transmitir en cualquier momento
  2. Existen mecanismos que pueden resolver la contienda por los medios
  3. La tecnología implementada es CSMA/CD para redes Ethernet 802.3 y CSMA/CA para redes inalambricas 802.11
  4. Acceso especifico para redes donde el radio de alcance no supera el kilometro, tratandose de un establecimiento cerrado
